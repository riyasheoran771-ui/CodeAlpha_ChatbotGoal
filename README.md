# CodeAlpha_ChatbotGoal
A simple rule-based chatbot built in Python that responds to predefined user inputs like greetings and questions. It uses functions, if-elif conditions, loops, and input/output to simulate a basic conversation until the user exits by typing "bye".
# 🤖 Rule-Based Chatbot

## 📖 Project Overview
The Rule-Based Chatbot is a beginner-friendly Python project that simulates a simple conversation with users. It recognizes predefined text inputs and responds with appropriate replies using conditional statements. This project is designed to help beginners understand the fundamentals of Python programming and chatbot logic.

## ✨ Features
- Greets users with friendly responses
- Answers predefined questions
- Handles basic conversation
- Runs continuously until the user types **"bye"**
- Displays a default response for unknown inputs

## 🛠️ Technologies Used
- Python 3

## 📚 Concepts Covered
- Functions
- `if-elif-else` statements
- `while` loops
- User input and output
- String methods (`lower()`)

## 🚀 Getting Started

### Prerequisites
- Python 3.x installed on your computer

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/rule-based-chatbot.git
   ```
2. Navigate to the project folder:
   ```bash
   cd rule-based-chatbot
   ```
3. Run the program:
   ```bash
   python chatbot.py
   ```

## 💬 Sample Interaction
```
ChatBot: Hello! I am a simple chatbot.
Type 'bye' to end the chat.

You: hello
ChatBot: Hi! Nice to meet you.

You: how are you
ChatBot: I'm fine, thanks!

You: bye
ChatBot: Goodbye! Have a great day!
```
